## 为什么这个仓库更像 GitHub 首页

它不是一个“语录包”，而是一个真正可运行、可审阅、可二次分发的 Skill：

- **不是复读原话**：重点提炼“他如何判断”，不是“他说过什么”
- **不是浅层角色扮演**：既能模拟语言风格，也能做第三人称行为预判
- **不是黑箱**：附带 `6` 份调研文件，来源透明，方便继续更新
- **不是一次性 Prompt**：核心逻辑已固化进 `SKILL.md`，可以直接安装使用
- **不是英文-only 项目**：主 README 中文优先，同时提供独立 `README_EN.md`

## 快速开始

### 1）安装

发布到 GitHub 后，把占位符替换成你的真实仓库路径：

```bash
npx skills add yangbuyiya/trump-perspective-skill
```

### 2）直接使用

在 Claude Code 中输入：

```text
> 用特朗普视角分析当前中东局势
> 预测特朗普接下来会怎么对待伊朗问题
> 特朗普怎么看霍尔木兹海峡危机？
> 解释这条 Truth Social 背后的真实信号
> 切换到特朗普口吻，和我聊一轮关税战
```

### 3）它会自动选择工作模式

- 你问“他会怎么看 / 会怎么做”时，走 `分析师模式`
- 你说“切换到特朗普口吻”时，走 `角色扮演模式`

## 效果示例

### 示例 1：伊朗问题预判

**用户：** 预测特朗普接下来会怎么对待伊朗问题？

**Skill 的回答方向：**
- 先做高压威慑，抬高谈判开价
- 把制裁、军事存在、公开威胁都当筹码
- 一旦油价、市场、盟友或国内政治成本上升，转向可包装成“赢”的降温安排
- 即便实际退让，也会用“重新定义胜利”的方式表达

### 示例 2：海峡危机分析

**用户：** 特朗普怎么看最近的海峡问题？

**Skill 的回答方向：**
- 不优先从国际法和地区秩序出发
- 更可能把它看成 `能源价格问题 + 市场情绪问题 + 对伊朗施压杠杆 + 对内展示强硬`
- 核心判断不是“该不该打”，而是“怎样既显得最强，又不背上长期战争成本”

### 示例 3：角色扮演

**用户：** 切换到特朗普口吻，评价一下这轮关税谈判。

**Skill 的输出风格：**
- 强势、短句、先下结论
- 高频使用绝对化表达与重复强调
- 把复杂问题压缩成 `赢 / 输`、`强 / 弱`、`忠诚 / 背叛` 的对立结构

## 两种使用模式

### 模式 A：分析师模式

默认模式，适合：

- “特朗普接下来最可能怎么做？”
- “他这条发言到底是在威胁，还是在开价？”
- “他对伊朗、乌克兰、关税问题的真实优先级是什么？”

输出会重点包含：

- 匹配到的核心心智模型
- 最可能触发的决策启发式
- 概率分布与置信度
- 关键不确定变量

### 模式 B：角色扮演模式

适合：

- “切换到懂王口吻”
- “如果特朗普来谈这个 deal，他会怎么说？”
- “用特朗普风格写一段回应”

输出会重点包含：

- 第一人称表达
- Trump-style 节奏、重复、绝对词、绰号式叙事
- 在没有原话时，基于心智模型做合理推断，而不是生硬拼贴语录

## 能力清单

### 1）核心心智模型

Skill 内置 `6` 个高频框架：

- `Everything Is A Deal`：万物皆交易
- `Truthful Hyperbole`：有效夸大
- `Unpredictability As Power`：不可预测性即权力
- `Victimhood As Fuel`：受害者叙事即燃料
- `Zero-Sum Winning`：零和赢
- `Audience First, Reality Second`：观众第一，现实第二

### 2）决策启发式

Skill 内置 `8` 条判断规则，包括：

- 最大化初始要价
- 威胁是筹码，不是政策
- 识别让步触发器
- 忠诚优于能力
- 个人化一切
- 永不认错，只重新定义胜利
- 面对攻击先反击
- 用 Roy Cohn 法则处理法律危机

### 3）表达 DNA

角色模拟时会尽量保留这些特征：

- 极短句
- 高频绝对词和最高级
- 重复关键词强化情绪
- 用绰号、贴标签、转移焦点制造叙事优势
- 常见结构：`Everybody knows` / `A lot of people are saying` / `We'll see what happens`

## 为什么它不像普通 Prompt

普通 Prompt 往往只能做到“模仿语气”。

这个 Skill 进一步把特朗普拆成了更稳定的结构：

| 层次 | 这里提炼了什么 |
|---|---|
| 怎么想 | 心智模型、世界观、判断框架 |
| 怎么做 | 决策启发式、让步触发器、升级逻辑 |
| 怎么说 | 表达 DNA、节奏、词汇偏好 |
| 什么不做 | 价值偏好、反模式、厌恶项 |
| 哪些地方不可靠 | 诚实边界与不确定性说明 |

所以它更适合做：

- 行为预测
- 谈判推演
- 舆论分析
- 风格模拟
- 多轮对话中的一致性输出

## 诚实边界

这个项目的价值在于“高质量推断”，不在于“全知全能”。

请务必注意：

1. **公开声明 ≠ 真实意图**
2. **不可预测性有时是策略，有时也可能是真随机**
3. **市场、盟友、国内政治约束会动态改变其动作**
4. **它更擅长谈判、传播、权力和行为预判，不等于能解释全部意识形态问题**
5. **它不是实时情报系统，结论受研究时间截点限制**

如果一个角色 Skill 从不告诉你它的局限，它往往也不值得信任。
